# 개인정보처리방침

## 간격 (Interval)

**최종 수정일**: 2025년 12월 16일

---

### 1. 개요

"간격"(이하 "앱")은 사용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보처리방침은 앱이 사용자 정보를 어떻게 처리하는지 설명합니다.

### 2. 수집하는 정보

**본 앱은 사용자의 개인정보를 수집하지 않습니다.**

앱 사용 과정에서 생성되는 모든 데이터(습관 기록, 타이머 설정 등)는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

### 3. 데이터 저장

- 앱 설정 및 사용 기록은 사용자 기기의 로컬 저장소에만 저장됩니다
- 저장된 데이터는 앱을 삭제하면 함께 삭제됩니다
- 데이터는 암호화되어 안전하게 보관됩니다

### 4. 제3자 공유

본 앱은 어떠한 사용자 데이터도 제3자와 공유하지 않습니다.

### 5. 앱 권한

앱은 다음 권한을 요청할 수 있습니다:

| 권한        | 용도                                           |
| ----------- | ---------------------------------------------- |
| 알림        | 사용자가 설정한 간격에 따른 리마인더 알림 발송 |
| 정확한 알람 | 정확한 시간에 알림을 전달하기 위함             |

이 권한들은 앱의 핵심 기능 제공을 위해서만 사용되며, 데이터 수집 목적으로 사용되지 않습니다.

### 6. 아동 개인정보

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

### 7. 개인정보처리방침 변경

본 방침이 변경될 경우, 앱 내 공지 또는 본 페이지를 통해 안내드립니다.
