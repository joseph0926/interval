지금 말고, 조금 있다가

### 프로모션 텍스트 (170자 이내)

```
끊으라고 강요하지 않아요. 나쁜 습관과의 '간격'을 조금씩 벌려보는 연습부터 시작해요. 오늘 5분만 미뤄보세요.
```

### 설명 (4000자 이내)

```
"지금 말고, 조금 있다가."

간격은 나쁜 습관을 끊게 해주는 앱이 아닙니다.
습관과의 '간격'을 조금씩 벌려주는 심리 타이머입니다.

━━━━━━━━━━━━━━━━━━━━

◆ 끊기가 아닌, 간격 늘리기

"당장 끊어!" 대신 "조금만 있다가 해볼까?"
강제로 막지 않고, 선택권을 드립니다.
작은 성공을 계속 쌓아가며, 자연스럽게 습관과 멀어지세요.

◆ 비난 없는 기록

목표를 못 지켜도 괜찮아요.
"실패"가 아니라 "오늘 패턴을 하나 더 발견했어요"라고 말해드립니다.
모든 기록은 당신을 이해하기 위한 데이터입니다.

◆ 한 손, 3초, 3탭

"지금 했어요" → 최대 3탭 안에 기록 완료.
바쁠 때는 시간만 빠르게, 여유가 있을 때는 30초 코칭과 함께.
당신의 상황에 맞춰 선택하세요.

━━━━━━━━━━━━━━━━━━━━

◆ 지원하는 습관 모듈

• 흡연 관리 - 담배와의 간격을 벌려보세요
• SNS 관리 - 스마트폰 중독에서 벗어나세요
• 카페인 관리 - 커피 섭취 간격을 조절하세요
• 집중 모드 - 뽀모도로처럼 집중 시간을 관리하세요

━━━━━━━━━━━━━━━━━━━━

◆ 주요 기능

• 습관 간격 타이머
  - 다음 행동까지 남은 시간 확인
  - 목표 간격 직접 설정

• 습관 기록
  - 원탭으로 간편하게 기록
  - 이유 선택 (스트레스, 습관, 지루함 등)

• 30초 멈춤 코칭
  - 조기 행동 시 잠시 멈추고 생각해보기
  - 5분만 더 미뤄보기 옵션

• 통계 & 인사이트
  - 주간 평균 간격 확인
  - 시간대별 패턴 분석
  - 미룬 시간 누적 확인

• 개인 동기 부여
  - 나만의 이유 한 줄 설정
  - 작은 성공 축하 메시지

━━━━━━━━━━━━━━━━━━━━

◆ 이런 분께 추천해요

• 나쁜 습관을 줄이고 싶은 분
• 당장 끊기보다 조금씩 줄이고 싶은 분
• 습관 패턴을 파악하고 싶은 분
• 비난 없이 기록하고 싶은 분
• 작은 성공으로 자신감을 쌓고 싶은 분

━━━━━━━━━━━━━━━━━━━━

오늘 나쁜 습관과 5분의 거리를 벌려보세요.
그 5분이 내일의 10분이 됩니다.

지금 시작하세요. 간격과 함께.
```

## 스크린샷 가이드

### 필수 스크린샷 (각 스토어 5-8장)

1. **홈 화면** - 타이머와 오늘 요약 카드
2. **습관 기록** - 기록 버튼과 이유 선택
3. **30초 코칭** - 멈춤 타이머 화면
4. **통계 화면** - 주간 그래프와 인사이트
5. **설정 화면** - 목표 간격 설정
6. **온보딩** - 앱 철학 설명

### 권장 텍스트 오버레이

1. "지금 말고, 조금 있다가"
2. "3탭으로 간편하게 기록"
3. "30초만 멈춰볼까요?"
4. "나의 습관 패턴 한눈에"
5. "오늘 5분의 거리를 벌렸어요"

## 개인정보 처리방침

스토어 등록 전 개인정보 처리방침 페이지 필요:

- 수집 정보: 습관 기록, 목표 설정, 사용 패턴
- 저장 위치: 로컬 기기 + 서버 (계정 연동 시)
- 제3자 제공: 없음
